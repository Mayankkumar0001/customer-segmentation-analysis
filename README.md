# Customer Segmentation Analysis

## Overview
This project focuses on segmenting customers based on their annual income and spending behavior.
The goal is to help businesses understand customer groups for targeted marketing.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed
- Exploratory Data Analysis (EDA)
- Age distribution analysis
- Income vs Spending visualization
- Customer clustering using K-Means
- Cluster visualization

## Dataset
Sample mall customer dataset used for segmentation analysis.

## Key Insights
- Identified multiple customer segments
- High-income high-spending customers detected
- Low-income low-spending customers detected
- Useful for marketing and customer targeting strategies
